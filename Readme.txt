Nearby Project

Описание
	На текущий момент этот проект предназначен для реализации алгоритма хеширования координат. Основная цель — разделить географическую область на квадранты для быстрого поиска объектов, находящихся рядом с пользователем (в радиусе 2 км).
	В дальнейшем планируется доработка до полноценно работающего сервиса с работающим интерфейсом и стабильной + защищенной серверной части.

Структура проекта:
	Geo.go: Содержит основную логику хеширования.
        - Функция HashCoords рекурсивно делит заданную область на четыре части (upperLeft, upperRight, downLeft, downRight) в зависимости от указанной точности (precision)
		- Функция FindNeighborsForHash (в разработке) предназначена для поиска соседних секторов для заданного хеша
	Main.go: Точка входа в приложение.
		- Инициализирует рабочую область (Area) размером 16x16
		- Генерирует хеш-карту (hashMap), где ключом является строковый хеш, а значением — список координат, попадающих в этот сектор

	User.go
		- Файл для логики, связанной с пользователем. В текущей версии пуст.
